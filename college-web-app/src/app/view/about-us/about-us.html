About Our Website

The new college website is designed with clean and modern HTML5 structure and elegant CSS3 styling.

Its homepage features a large banner with the college logo and a welcoming message to students and visitors.

The navigation bar at the top provides quick links to Admissions, Departments, Courses, and Contact pages.

CSS flexbox and grid layouts are used to keep all sections responsive and well-aligned on any device.

The About Us page introduces the college’s mission, vision, and history with soft background colors and readable fonts.

The Departments section displays cards for each department, styled with hover effects using CSS transitions.

A separate page lists all courses, using tables and colored headings for clarity and organization.

The Contact page includes a simple HTML form for inquiries, styled with rounded input fields and subtle shadows.

The entire website uses a consistent color theme — blue and white — symbolizing trust and knowledge.

With proper HTML semantics and CSS responsiveness, the college website looks professional, accessible, and user-friendly.
